서빙 로봇 구성

명칭 및 기능

1
비상 정지 버튼 : 로봇 긴급 정지
2
무게 감지 센서 : 적재량을 감지하여 자동 복귀
3
전원 스위치 : 로봇의 전원 ON/OFF
4
충전기 포트 : 충전기를 연결하여 로봇을 충전

  • 주행 전 시작 위치를 확인해 주세요.
  • 주기적으로 카메라 센서를 청소해 주세요.
  • 경로 내 구조물 추가 시, 맵을 수정해 주세요.

참고 서빙 로봇 이용 시 위 사항을 준수해 주세요.

서빙 조작

1
로봇 화면 내 [서빙]을 터치해 주세요.
2
테이블 지정 후 [배송]을 터치해 주세요.
3
상품 수령 후 해당 종류에 따라 [출발] 버튼 터치 시 다음 장소 또는 대기 장소로 복귀해요.
참고 설정에서 자동 출발 설정을 할 수 있어요.

오더 태블릿 로봇 호출

1
먼저 호출된 테이블 순서로 서빙 로봇 이동해요.
2
우측 상단 테이블 번호를 빠르게 5번 터치해 주세요.
3
비밀번호 입력 후 [확인]을 터치해 주세요.
4
먼저 호출된 테이블 순서로 서빙 로봇 이동해요.
